# links-helper-bot

<p align="center">
  <img src="./links-helper-bot.jpg" alt="Links Helper Bot" width="320">
</p>

links-helper-bot is a Telegram bot for saving useful links and coming back to them later.

Send the bot an article, video, documentation page, product link, or any other URL, and it will keep it in your personal reading list. When you have time, ask the bot for a random saved link and return to something worth reading instead of losing it in chats, browser tabs, or bookmarks.

## What It Can Do

- Save links sent directly to the bot.
- Keep links separately for each Telegram user.
- Return a random unread link.
- Show recently saved links.
- Search through saved links.
- Delete links you no longer need.
- Show simple stats for saved, unread, and read links.

The active development version lives in the `develop` branch.
